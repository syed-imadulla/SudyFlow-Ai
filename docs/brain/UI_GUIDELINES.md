# UI Guidelines

## Theme
- **Global Theme**: Dark
- **Accent Color**: Purple

## Layout & Aesthetics
- **Rounded Corners**: Generous usage on interactive elements and containers.
- **Spacing**: Consistent padding and margin scales to ensure breathing room.

## Components
- **Animation Style**: Smooth, fast, and subtle transitions (e.g. hover lifts, slight scale-ins).
- **Cards**: Surface-level containers with distinct borders and subtle glowing shadows on hover.
- **Buttons**: Clear visual hierarchy, pill or slightly rounded shapes, distinct active/hover states.

## Typography
- **Design Principles**: High contrast for readability on dark backgrounds, using system or modern sans-serif fonts to maintain a sleek, focus-oriented look.
